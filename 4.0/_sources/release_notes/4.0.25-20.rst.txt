================================================================================
*Percona Server for MongoDB* 4.0.25-20
================================================================================

:Date: June 17, 2021
:Installation: `Installing Percona Server for MongoDB <https://www.percona.com/doc/percona-server-for-mongodb/4.0/install/index.html>`_

Percona Server for MongoDB 4.0.25-20 is an enhanced, open source, and highly-scalable database that is a
fully-compatible, drop-in replacement for `MongoDB 4.0.25 Community Edition <https://docs.mongodb.com/v4.0/release-notes/4.0/#jun-10-2021>`_.
It supports MongoDB 4.0.25 protocols and drivers.
